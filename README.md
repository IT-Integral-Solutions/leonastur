# leonastur# 🦁 Leonas Tur — Landing Oficial

Sitio web oficial de **Leonas Tur**, agencia de viajes especializada en experiencias personalizadas, asesoramiento integral y planificación estratégica de viajes.

---

## 🌍 Descripción

Este repositorio contiene la landing page de Leonas Tur, diseñada para:

* Captar leads de potenciales clientes
* Presentar destinos y servicios destacados
* Generar contacto directo vía WhatsApp
* Servir como base para futuras integraciones con sistemas CRM

---

## 🚀 Tecnologías utilizadas

* HTML5
* CSS3
* Imágenes optimizadas (WebP, PNG, JPG)
* Estructura lista para despliegue en:

  * Cloudflare Pages
  * GitHub Pages

## ⚙️ Instalación y uso

1. Clonar el repositorio:

```

```

2. Abrir el proyecto:

```
cd leonastur
```

3. Ejecutar localmente:

👉 Abrir `index.html` en el navegador
